# Convertie

A sleek, modern converter that intelligently detects input types and converts them to your desired format — instantly.

## ✨ Features

- **Smart Auto Detection** — Automatically recognizes common data formats (CBOR, Hex, Base64, Text, etc.)
- **One-click Conversion** — Convert between multiple formats with a single click
- **Auto Convert** — Paste into the input box and it automatically converts based on your chosen "From → To" settings
- **Real-time Feedback** — Shows byte/character count and format details
- **Clipboard Integration** — Designed for quick copy-paste workflows

### Supported Formats (examples)
- **CBOR** (Concise Binary Object Representation) — including diagnostic notation
- **Hex** (with or without spaces)
- **Text** / UTF-8
- More other formats

## How to Use

1. **Paste or type** your data into the input box and it will converts it to the first detected output type
2. **Change** the output type and it will convert the input to the desidred output instantly
3. **Setup** an auto conversion workflow and the app will convert the desired input type if it finds it in clipboard once the input textbox is focused.
